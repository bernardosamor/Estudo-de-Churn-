# Análise de Churn em Empresa SaaS (Excel)

> **Resumo:** Estudo focado na identificação de padrões comportamentais que guiam a desistência de clientes, mapeando os principais gatilhos para otimização da retenção.

---

### 💡 O Problema de Negócio
Em modelos de receita recorrente, o churn impacta diretamente a saúde financeira. O desafio aqui foi entender: *Por que os clientes estão saindo e quais variáveis indicam que um cliente está prestes a cancelar?*

### 🛠️ O que compõe este projeto?
Como este é um repositório focado em análise via Excel, o arquivo disponível contém o fluxo completo:
* **Base Bruta:** Dados extraídos para análise.
  * **Aba de Hipóteses:**  Ideias que guiaram a análise, insights que pensei em buscar a partir de padrões de comportamento de empresas SaaS.
  * **Aba Churn:**  Levantei os dados de churn ciclo a ciclo para ter um overview geral da situação, e verificar momento de virada de retenção crítica (3 primeiros meses).
  * **Aba de Método de pagamento:** Verificação do principal gatilho para desistências.
* **Tabela Absoluta:** Visualização dos indicadores de forma mais completa, trazendo a retenção, desistência e a taxa de retenção de forma completa em uma só tabela.
* **Análise preditiva:** Listando os clientes que estão mais propensos a entrar em Churn baseado no último ciclo registrado.

---

### 📈 Principais Insights
* **Métodos de Pagamento:** Devido a necessidade manual de confirmar o pagamento, o boleto eletrônico impulsiona os casos de Churn. **Incentivar a assinatura do serviço via cartão de crédito** (bonus de 15% de desconto nos 3 primeiros meses) **é um forte plano de ação**.
* **Fortalecer o onboarding da empresa:** Considerando que não se adaptar ao produto pode guiar a desistência, **front end intuitivo, bonificação ao uso constante da plataforma, surgem como planos de ação alternativos**. 
* **Posição da empresa no mercado de SaaS:** A empresa chega ao final do período avaliado com **7% de casos de churn ao ano**, valor consideravelmente abaixo do padrão do segmento, 13%, **indicando alta satisfação com o serviço**.
* **Redução dos casos no período crítico:** Entendendo que o churn asboluto é o ponto forte da empresa, antecipara estabilidade, **tentar reduzir o período crítico para novos usuários de 3 para 2 meses**.
